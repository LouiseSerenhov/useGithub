Use Github, step by step

An easy way to save your code, use version control and work in a team!
  • Step 1:

    - Create a Github account at https://github.com/

  • Step 2:

    - Open your terminal. You can search for it in the launchpad (on mac).

    - Try to find your way to the desktop. You can follow this guide to learn how.

    - Check if Git is installed (in terminal) by writing:

    git --version

    - If it is installed go to step 3.

    - Otherwise install git through https://git-scm.com/downloads

    - And then check if it´s installed with:

    git --version

  • Step 3:

    - Login to your Github account and press “repositories” and then the green button “New”.

    - Name the repository whatever you want (you can change it later or delete it).

    - Have “public” activated and press “initilaize this…”.

    - Ignore the last two buttons and press “Create repository”.

  • Step 4:

    - Now that you have a repository. Go to your profile and click on the repository you created.

    - Then press the green button in the right corner “Clone or download” and copy that url.

  • Step 5:

    - Go to your terminal and find your way to the desktop. You can follow this guide to learn how.

    You only need to learn "ls", "cd" and "pwd" right now.

    - When you are in the desktop at the terminal you can write the following in the terminal:

    git clone “the url you just copied”

  • Step 6 (option 1, good to do it this way):

    - Now you have the repository you created on your computor.

    - Try to put a html file in that folder and write something.

    - Then go to your terminal and make sure you are in that folder and write.

    git add .

    git commit –m “my first commit”

    git push

  • Step 6 (option 2, easier but a little bit cheating):

    - Now you have the repository you created on your computor.

    - Try to put an index.html file in that folder and write something.

    - Then go to the scissor (branch) at the left in Visuol Studio and write "my first commit" in the box.

    - Then press command + enter

    - Then press the three dots in the upper right corner (still in the left box) and press "push"

  • Step 7:

    - Go to your Github account and see that your repository now contains the new file (yay!).

    - Now you can add anything you want by using the commands from step 6:

    - It will take a few minutes before it works!
